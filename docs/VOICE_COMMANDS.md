# Voice Commands User Guide

Welcome to the Phone Log Voice Assistant! This guide covers all the voice commands you can use to manage your call records hands-free.

## Getting Started

1. Click the 🎤 microphone button to start voice input
2. Speak naturally - the assistant understands conversational language
3. Wait for the assistant to respond (both text and voice)

## Basic Commands

### Greeting the Assistant

| Say This | What Happens |
|----------|--------------|
| "Hi" / "Hello" | The assistant greets you back |
| "Good morning" | Context-aware greeting |
| "Hey there" | Casual greeting response |

### Getting Help

| Say This | What Happens |
|----------|--------------|
| "Help" | Shows all available commands |
| "What can you do?" | Lists assistant capabilities |
| "How do I use this?" | Displays usage guide |

### Learning About the Assistant

| Say This | What Happens |
|----------|--------------|
| "Who are you?" | Assistant introduces itself |
| "What's your name?" | Reveals assistant's name |
| "Tell me about yourself" | Full introduction with background |
| "What's your full name?" | Shows complete name (for personas) |

## Call Management Commands

### Viewing Calls

| Say This | What Happens |
|----------|--------------|
| "Show my calls" | Lists all call records |
| "List all calls" | Displays complete call log |
| "View my recent calls" | Shows calls (most recent first) |
| "What calls do I have?" | Lists call records |

### Searching Calls

| Say This | What Happens |
|----------|--------------|
| "Search for Alice" | Finds calls with "Alice" |
| "Find calls from 555-1234" | Searches by phone number |
| "Look up Bob" | Searches by contact name |
| "Any calls from Mom?" | Finds calls from "Mom" |
| "Did John call?" | Checks for calls from John |

### Adding Calls

| Say This | What Happens |
|----------|--------------|
| "Add a new call" | Opens the add call form |
| "Log a call" | Prompts for call details |
| "I had a call with Alice" | Initiates call logging |
| "Record a phone call" | Opens add call interface |
| "I received a call from Bob" | Starts logging a call |

### Deleting Calls

| Say This | What Happens |
|----------|--------------|
| "Delete call 5" | Removes call record #5 |
| "Remove record 10" | Deletes call #10 |
| "Erase call ID 3" | Removes call with ID 3 |

### Statistics

| Say This | What Happens |
|----------|--------------|
| "How many calls do I have?" | Shows total call count |
| "Give me statistics" | Displays call breakdown |
| "Call stats" | Shows incoming/outgoing/missed counts |
| "Count my calls" | Total call count |

## Tips for Better Recognition

### Speak Clearly
- Pause briefly before speaking
- Speak at a normal pace
- Enunciate numbers clearly ("five" not "5")

### Natural Language Works Best
The assistant understands natural conversation:
- ✅ "Show me all my calls from last week"
- ✅ "Did anyone named Smith call me?"
- ✅ "I need to log a missed call"

### Phone Numbers
When saying phone numbers:
- Say digits individually: "five five five, one two three four"
- Or naturally: "555-1234"

## Troubleshooting

### Voice Not Recognized
1. Check microphone permissions in browser
2. Ensure microphone is not muted
3. Try speaking more clearly
4. Check for background noise

### Browser Compatibility
Voice recognition works best in:
- Google Chrome (recommended)
- Microsoft Edge
- Safari (limited support)

### If Commands Aren't Working
Try rephrasing:
- Instead of "gimme calls" → "show my calls"
- Instead of "kill record 5" → "delete call 5"

## Privacy & Security

- Voice processing happens locally in your browser
- No audio is sent to external servers
- Call data is stored locally in your installation
- You can disable voice features anytime
